# Lahja Brand Guide

## Colors
Primary Green: #1F6F54  
Sand: #E6D5B8  
Charcoal: #1F1F1F  
Accent Red: #8C3A2B  

## Fonts
English: Montserrat Bold  
Arabic: Cairo Bold  
Body: Open Sans Regular  

## Style Rules
- Calm, minimal UI
- Patterns used only as dividers/background accents
- No heavy shadows
- Flat colors only
